# 定位框架参考

## 定位陈述模板

```
经典版：
针对 ___（目标用户），___（产品名）是一个 ___（品类），能 ___（核心价值），因为 ___（独特的理由）。

Before/After/Bridge（BAB）：
之前：___（痛苦的现状）
之后：___（理想状态）
桥梁：___（你的产品）如何帮用户从这里到那里

PAS（Problem-Agitate-Solution）：
问题：___（用户的痛苦）
放大：___（为什么现在不解决会越来越麻烦）
方案：___（你的方案为什么不同）
```

## 定位地图坐标轴

| 坐标轴 | X 轴左 | X 轴右 | Y 轴下 | Y 轴上 |
|---|---|---|---|---|
| 价值轴 | 低价 | 高价 | 基础功能 | 高级功能 |
| 用户轴 | 个人/小团队 | 企业 | 单点工具 | 平台 |
| 体验轴 | 复杂/高学习门槛 | 简单易用 | 功能少 | 功能全 |

> 白色空间 = 你的机会。如果所有竞品都在右上角（高价全功能），左下角（低价单点）就是你的切口。

## 定位地图和使用

定位地图帮助可视化竞品分布，发现被忽视的市场空间。

**操作步骤：**
1. 选择一组买方相关的坐标轴（从上方选择或自定义）
2. 标记直接竞品、间接替代品和你的当前位置
3. 识别白色空间、拥挤集群和坐标轴本身的盲区
4. 选择一个与能力匹配、未被充分服务且可防御的位置

## 差异审计："Only-We" 测试

问自己——**只有你的产品能说 ___（因为独特的原因）**。如果竞品也能说同样的话，那就不是真正的差异。

## 定位防错清单

| 反模式 | 修复 |
|--------|------|
| 把目标位置当现状 | 先记录当前定位，再规划目标定位 |
| 用虚荣指标做坐标轴 | 坐标轴必须来自用户语言验证 |
| 只分析直接竞品 | 包含间接工具、Excel、现状 |
| 一次性地图不更新 | 每季度或竞品有大动作时刷新 |
