# 竞品 Battlecard 模板

需要持续跟踪某竞品时使用此模板。

```
## 竞品档案：___

**基础信息**
- 产品：___ | 定价：___ | 目标用户：___
- 成立时间：___ | 团队规模（估算）：___
- 流量（估算）：___ 月访问量 | DA（估算）：___

**他们做得好**
- 获客上：___（什么渠道/策略有效）
- 产品上：___（什么功能用户离不开）
- 口碑上：___（用户夸他们什么）

**他们的短板（来自评论挖掘）**
- ___（来自用户评论，附来源）
- ___（来自用户评论，附来源）

**你的差异点**
- 你做什么和他们不同：___
- 用户为什么可能选你：___
```
